cara menjalankan web app
1. ekstrak terlebih dahulu folder
2. buka terminal (command prompt/windows powershell)
3. pergi ke path folder source code (dengan perintah cd)
4. pergi ke folder web/scripts/ (dengan perintah cd)
5. ketik activate.bat lalu enter
6. kembali ke folder awal dengan mengetik "cd .." dua kali
7. ketik "streamlit run streamlit-app.py"
8. tunggu hingga web browser terbuka atau masuk ke IP yang tersedia pada terminal setelah aplikasi berjalan
